# portfolio


To run the app:

- download repo
- yarn install
- yarn serve

Simple Vue demo page using components, Router, Vuex.

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
